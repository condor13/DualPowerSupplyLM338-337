Linear Power Supply for Synthesizers (+12V, -12V, +5V)
A high-quality, low-noise linear power supply designed specifically for analog synthesizers.

Features
Triple Rail Output: +12V, -12V, and +5V rails.
High Current Capability: Equipped with LM338 (5A) for positive rails and LM337 (1.5A) for the negative rail.
Adjustable Precision: Multi-turn trimmers allow for exact voltage calibration (e.g., exactly 5.00V).
Robust Protection: Includes 1N4007 back-EMF protection diodes on all stages.

Hardware Specifications
Input: Transformer with 18V-0-18V and 10V secondaries.
Regulation: LM338T (Pos), LM337T (Neg).
Thermal: Designed for external heatsink mounting with electrical isolation.
